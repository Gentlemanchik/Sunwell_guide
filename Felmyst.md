# Пророк Скверны #

Третий босс на Плато Солнечного колодца. Считается самым лёгкий энкаунтером среди всех шести. Данный энкаунтер похож на бой с Ночной Погибелью из каражана. Тоже есть земная фаза, на которой нужно просто стоять и исполнять свои роли (хотя мб и нет, в зависимости от тактики). Есть лётная фаза, на которой дракон будет поливать рейд лужей и будут спавнится много аддов-скелетов. Ещё данный бой чуть-чуть напоминает Сапфирона из Наксрамаса. По рейду постоянно будет будет тикать урон от ауры (но только на наземной фазе), критически важно диспелить (у Сапфирона нужно было жекурсить).
